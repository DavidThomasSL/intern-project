Welcome to Clonage!

Like Fibbage, but better...

## To install

1) Get Clonage from GitHub

2) Install node

3) Run "npm install" to install dependancies 

## Running the server

1) Run "node server.js" from the same directory as this readme
2) Go to http://intern-project.herokuapp.com/
